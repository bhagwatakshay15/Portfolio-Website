Income-Gender analysis
Data analysis in R

This is a brief summary of a data analysis project on gender-based income disparity. Using the National Longitudinal Survey of Youth 1997 (NLSY97) dataset, the project aimed to see if a measurable income gap exists between men and women and how other factors like education and criminal history affect this. The analysis, performed using R, concluded that a persistent income gap does exist. The study found that while other variables influence income, gender remains a strong predictor of a person's income.
